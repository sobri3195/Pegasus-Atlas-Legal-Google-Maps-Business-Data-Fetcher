# Pegasus Atlas - User Guide

## Getting Started

### Installation

1. Download the latest release for your platform
2. Install the application
3. Launch Pegasus Atlas

### First Search

1. Enter a keyword (e.g., "restaurants in Jakarta")
2. Configure search options:
   - **Headless Mode**: Run browser in background (faster)
   - **Simulate Human Behavior**: Add random delays and mouse movements
   - **Max Results**: Limit number of results
   - **Delay**: Time between actions (milliseconds)
3. Click "Start Search"

## Features

### Single Search

Use single search for one-off queries:
- Enter one keyword
- Get immediate results
- Perfect for testing and small queries

### Batch Search

Use batch mode for multiple keywords:
1. Check "Batch Mode"
2. Enter multiple keywords (one per line)
3. Click "Start Batch Search"
4. Monitor progress in real-time

Example batch input:
```
restaurants in Jakarta
cafes in Surabaya
hotels in Bali
```

### Data Cleaning

Remove duplicates and normalize data:
1. Collect your data
2. Click "Clean & Deduplicate"
3. System will:
   - Remove exact duplicates
   - Normalize addresses
   - Standardize phone numbers

### Export Options

#### CSV Export
- Plain text format
- Easy to open in Excel/Google Sheets
- Best for simple analysis

#### Excel Export
- Formatted spreadsheet
- Headers and styling included
- Best for professional reports

#### JSON Export
- Machine-readable format
- Best for developers
- Easy to import into other tools

#### PDF Export
- Summary report format
- First 100 records only
- Best for presentations

## Tips & Best Practices

### Performance
- Use headless mode for faster searches
- Increase delay if experiencing issues
- Batch search is more efficient for multiple keywords

### Data Quality
- Use specific keywords for better results
- Clean data regularly to remove duplicates
- Review results before exporting

### Legal & Ethical Use
- Only collect public information
- Respect website terms of service
- Implement reasonable delays
- Use for legitimate purposes only

### Troubleshooting

**Browser doesn't open:**
- Try unchecking headless mode
- Check system resources
- Restart application

**No results found:**
- Try different keywords
- Increase max results limit
- Check internet connection

**Application crashes:**
- Reduce max results
- Increase delay between actions
- Clear database and try again

## Keyboard Shortcuts

- `Ctrl/Cmd + R` - Refresh results
- `Ctrl/Cmd + E` - Export data
- `Ctrl/Cmd + K` - Focus search input
